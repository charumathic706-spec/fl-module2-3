# Dataset

## Option A — Kaggle Credit Card Fraud (recommended)
Download: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
Place at:  BATFL/data/creditcard.csv

284,807 transactions, 492 frauds (0.172% imbalance ratio).

## Option B — Synthetic (no download)
Add --synthetic flag to any command. Generates 10,000 samples automatically.
